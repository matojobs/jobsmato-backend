# Config

Configuration files (not secrets; use `.env` for secrets).

| Path | Description |
|------|-------------|
| `nginx/` | Nginx configs. When using the Docker nginx service, mount `./config/nginx/nginx.conf` as `/etc/nginx/nginx.conf`. |
| `jobsmato-folder-config.json` | Google Drive Jobsmato folder config; generated by `scripts/create-jobsmato-folder.js`. |

`env.example` stays at repo root; copy to `.env` and fill in values.
